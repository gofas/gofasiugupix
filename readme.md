# Gofas iugu Pix

Módulo de gateway de pagamento para WHMCS que integra cobranças via Pix pela API iugu. Desenvolvido pela Gofas Software.

## Funcionalidades

- Geração de cobranças Pix via API iugu
- QR Code dinâmico na área do cliente
- Baixa automática de fatura

## Requisitos

- WHMCS 7.x ou superior
- PHP 8.x
- Conta iugu com Pix habilitado e API Token

## Instalação

1. Copiar `modules/gateways/` para o `modules/gateways/` do WHMCS
2. Ativar e informar API Token
